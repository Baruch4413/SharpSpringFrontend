# notes-frontend

## Project setup
Create a .env file with the following content:
```
BACKEND_SERVER_URL="<YOUR_SERVER_URL>"
```
Then run
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
